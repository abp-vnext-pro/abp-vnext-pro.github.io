前言

创建付费版本的后端，需要创建token。

  • 打开github右上角Settings

  • 点击Developer Settings

  • 选择Fine-grained Tokens

  • 设置相关信息

  • 设置权限

  • 点击保存

  • 通过token登录cli

bash
lion.abp login -token 你的token
1
  • 登录成功之后就可以创建付费的项目了。
